GridInfo类

网格信息

 

构造函数

构造函数：public GridInfo();

 

字段

名称

类型

说明

Coord

byte

【主刻度坐标注记】是否添加主刻度坐标注记：0 不添加，1 添加

Cxy

byte

格网划分类型： 0:均分网格,1:整倍标注,2:网距划分

Dx

double

【网格划分】网距(网线值约数)(x方向)，米(经纬网：度分秒)

Dy

double

【网格划分】网距(网线值约数)(y方向)，米(经纬网：度分秒)

Ex

double

【网格划分】结束网线值的约数(x方向)，米(经纬网：度分秒)

Ey

double

【网格划分】结束网线值的约数(y方向)，米(经纬网：度分秒)

Graduate

byte

【主刻度线】是否添加刻度线：0 不添加，1 添加

GraduateClr

int

【主刻度线】刻度线颜色

GraduateLen

float

【主刻度线】刻度线长

GraduatePos

byte

【主刻度线】刻度线位置：0 框内，1 框外

GraduateWid

float

【主刻度线】刻度线宽

Line

byte

【网格线】网格线类型：0 无网线，1 十字，2 实线

LineClr

int

【网格线】网线颜色

LineCrossLen

float

【网格线】十字线长（Line等于1时有效）

LineEx

byte

【网格线】网格线是否外延：0网格线不外延，1 网线外延与相邻图廓线相连接（此时没有主刻度）

LineWid

float

【网格线】网线线宽

MidNote

byte

【网格间注记】是否注记添加网格间注记：0 不添加，1 添加

MidNotePos

byte

【网格间注记】网格间注记位置：0 框外，1 框内

MidNoteX

byte

【网格间注记】x注记类型：0 "1、2、3..."， 1 "A、B、C..."(最多26个)，2 "①、②、③..."(最多10个)

MidNoteY

byte

【网格间注记】Y注记类型：0 "1、2、3..."， 1 "A、B、C..."(最多26个)，2 "①、②、③..."(最多10个)

Nx

short

【网格划分】网格个数(x方向)

Ny

short

【网格划分】网格个数(y方向)

SubCoord

byte[]

【支刻度尺】每级支刻度是否添加注记：0 不添加，1 添加

SubDigitX

short[]

【支刻度尺】每级支刻度X注记小数位数

SubDigitY

short[]

【支刻度尺】每级支刻度Y注记小数位数

SubGraduate

byte[]

【支刻度尺】每级支刻度线个数（目前最多支持3级支刻度，=0 终止符）

SubGraduateClr

int[]

【支刻度尺】每级支刻度线颜色

SubGraduateLen

float[]

【支刻度尺】每级支刻度线长

SubGraduateWid

float[]

【支刻度尺】每级支刻度线宽

SubOnGridLine

byte

实线网格线上是否添加支刻度短线：0 无，1 添加1级支刻度，2 添加2级支刻度...

Sx

double

【网格划分】起始网线值的约数(x方向)，米(经纬网：度分秒)

Sy

double

【网格划分】起始网线值的约数(y方向)，米(经纬网：度分秒)

Type

byte

【基本参数】坐标网类型：0 直角坐标网（方里网），1 地理坐标网（经纬网）
