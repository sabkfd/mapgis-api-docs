CheckGraAttCoe(

   SFeatureCls checkVcls,

   string def,

   ObjectIDs iDPartList,

   SFeatureCls mapVcls,

   string mapDef,

   int mapType,

   int cohType,

   int sitType,

   List<double> dToleranceArray,

   int lToleranceLen,

   out ObjectIDs rtnOIDS

)

bool

成功返回true，失败返回false

图形属性一致性检查  

checkVcls：待检查的简单要素类

def：待检查类的表达式

iDPartList：待检查类中的要素OID列表（若为空，全检查）

mapVcls：映射的简单要素类

mapDef：映射类的表达式

mapType：0 - 1:1，1 - 1:n，2 - 一个图元多个表达式对应多个图元一个表达式

cohType：一致性类型(1：表示相等，若按照表达式计算的值为浮点类型，此时参数dTolerance才有意思，表示容限 2：大于 3：小于)

sitType：坐落类型：1-B落在A上，2-B落在A的空间范围内，并不一定在B上,比如带洞的区(落在带洞的区的洞里)，3- A落在B上

dToleranceArray：比较条件语句需要的容差,例如两个数进行比较相等时如果两个数的差小于多少代表相等

lToleranceLen：比较条件语句的容差个数,即array大小

rtnOIDS：【输出】返回待检查类中不符合检查要求的OID
