属性改变事件 (从MapGIS.GeoMap.MapLayer继承)

Display  ：获取自定义绘图对象

Layer：获取图层对象

PropertyName：获取属性名称

PropertyValue：获取属性值

 

示例

Rotate 方法示例

C#

using MapGIS.GeoMap;

using MapGIS.GeoObjects.Geometry;

using MapGIS.GISControl;

 

//变量定义

Document doc = null;

Maps maps = null;

Map map = null;

MapLayer layer = null;

Transformation trans = null;

//获取图层

doc = new Document();

doc.Open(@"F:\10\MapGIS10.2\Sample\wuhan.xml");

maps = doc.GetMaps();

map = maps.GetMap(0);

layer = map.get_Layer(1);

trans = this.mapControl1.Transformation;

//显示地图

this.mapControl1.ActiveMap = map;

this.mapControl1.Restore();

//旋转图元

layer.Rotate(100, new Dot(114.375, 30.57), 30, true, trans);

//闪烁该图元

this.mapControl1.StartFlash(layer, 100, 1000, 6, 5);

备注

1、此方法将修改数据本身，即改变图元的实际位置；

2、旋转角度，旋转完整一周是360度，即赋值360，正值表示以旋转中心为圆心，顺时针方向旋转，负值则相反；

3、逻辑坐标标志，其与旋转中心点对应，当旋转中心点采用逻辑坐标，则逻辑坐标标志则为true（true表示：旋转中心坐标为逻辑坐标），trans参数可置空，反之若采用设备坐标，则为false（false表示：旋转中心坐标为非逻辑坐标），例如“layer.Rotate(100, new Dot(this.mapControl1.ViewWidth / 2, this.mapControl1.ViewHeight / 2), 30, false, trans);”，则trans参数须给定当前的坐标转换对象。
