UniqueTheme类

单值专题图，又称唯一值专题图

在单值专题图上，可以根据属性值（或特征）来绘制要素并识别它们，是用不同的颜色或图案表示属性表中指定字段的每一个不同的值，强调数据中的类别差异。比如在制作土地利用单值专题图时，可以用特定的颜色来表示每一种土地利用类型；在行政区划图中，可以对每个行政区设置不同的图形参数来体现各个区的大小范围。

注：单值专题图只支持整数型、字符串型、日期型的数据。

示例

Append 方法示例

C#

using MapGIS.GeoDataBase;

using MapGIS.GeoObjects;

using MapGIS.GeoObjects.Info;

using MapGIS.GeoMap;

 

 

//数据库服务器

Server srv = null;

//数据库

DataBase db = null;

//数据

SFeatureCls themeSFeature = null;

 

VectorLayer themeLayer = null;

 

//打开数据源

srv = new Server();

srv.Connect("mapgislocal", "", "");

db = srv.OpenGDB("demodata");

 

//打开用于统计的简单要素类

themeSFeature = (SFeatureCls)db.GetXClass(XClsType.SFCls);

themeSFeature.Open("平若县行政略图.WP", 0);

 

themeLayer = new VectorLayer(VectorLayerType.SFclsLayer);

 

if (themeSFeature != null)

    themeLayer.AttachData(themeSFeature);

 

//单值专题图

UniqueTheme uniqueTheme = null;

//单值专题图信息

UniqueThemeInfo uniqueInfo = null;

//区信息

RegInfo rInfo = null;

 

uniqueTheme = new UniqueTheme();

uniqueInfo = new UniqueThemeInfo();

rInfo = new RegInfo();

 

rInfo.Ovprnt = true;

 

//循环添加单值专题图项

for (int i = 0; i < 4; i++)

{

    rInfo.FillClr = 10 + i;

 

    uniqueInfo.GeoInfo = rInfo;

    uniqueInfo.Value = i.ToString();

    uniqueInfo.Caption = "第" + i + "个";

 

    //添加

    uniqueTheme.Append(uniqueInfo);

}

 

//单值专题图信息

uniqueInfo.IsVisible = true;

uniqueInfo.MaxScale = 0;

uniqueInfo.MinScale = 0;

 

//单值对应字段

uniqueTheme.Expression = "COLOR_INDEX";

uniqueTheme.Visible = true;

//专题图名称

uniqueTheme.Name = "平若县行政略图";//必须给

 

//添加专题图

themeLayer.Themes.Clear();

themeLayer.Themes.Append(uniqueTheme);

备注

1.此示例代码需要在VS工程里的引用下面添加如下组件，分别为MapGIS.GeoMap.dll,MapGIS.GeoObjects.dll,MapGIS.GeoDataBase.dll；

2.可以在平台安装目录下面的Program文件夹下面找到对应的组件。
