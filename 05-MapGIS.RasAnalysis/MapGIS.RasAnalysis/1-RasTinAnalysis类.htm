static RsBatchCalVolume(

   SFeatureCls TinObj,

   double zFactor,

   double Height,

   bool IsUpHeight,

   double CalDensity,

   SFeatureCls RegSFCls,

   int OutWay,

   out List<CalVolume_Stru> VolumeStruList,

   out List<int> TresList,

   string ResName

)

Int32

成功/失败 1/0

表面积、土方量计算 

TinObj：TIN对象

zFactor：Z值放大因子,无变形为1

Height：指定计算的平面高度

IsUpHeight：计算该指定高度的方式：IsUpHeight = TRUE-计算平面以上，IsUpHeight = FALSE-计算平面以下

CalDensity：区域密度

RegSFCls：区域简单要素类

OutWay:：计算结果的方式：0-输出到数组中，1-输出到文本中，2-输出到数组和文本中

VolumeStruList：区域土方量列表

TresList：所有区的要素ID序列

ResName：输出的结果文本文件绝对路径名称
