示例

SetAnalyzeInfo 方法示例

此处示例代码主要介绍了用户输入模式的洪水淹没分析功能的实现以及分析信息的设置；运行该示例需在Form窗体中添加场景显示控件SceneControl。

C#

 
using MapGIS.GeoDataBase;
using MapGIS.Scene3D;
using MapGIS.GeoObjects.Geometry;
using MapGIS.GeoDataBase.GeoRaster;
 
Server svr = new Server();
//连接数据源
svr.Connect("mapgislocal", "", "");
//获取场景数
int count= this.sceneCtrl.GetSceneNum();
Scene scene = new Scene();
if (count > 0)
{
    //获取场景
    scene = this.sceneCtrl.GetScene(0);
}
else
{
    //添加场景
    this.sceneCtrl.Append(scene);      
}              
 
 //创建栅格数据集对象
RasterDataSet rasSet = new RasterDataSet();
//打开栅格数据集
bool t = rasSet.Open(@"gdbp://MapGisLocal/sample/ras/高程数据", RasAccessType.RasAccessType_ReadOnly);
 
if (t)
{
    //创建地形图层对象
    TerrainLayer terrainLayer = new TerrainLayer();
 
    //关联数据
    terrainLayer.URL = rasSet.URL;
    terrainLayer.Name = rasSet.ClsName;
    terrainLayer.ConnectData();
 
    //添加地形图层到场景中
    scene.Append(terrainLayer);
    scene.ActiveLayer = terrainLayer;
    //洪水淹没演示
    FloodAnalyzeInfo floodAnalyzeInfo = new FloodAnalyzeInfo();
    //透明度(0-1.0之间有效)
    floodAnalyzeInfo.Alpha = 0.5;
    //淹没区域颜色
    floodAnalyzeInfo.FloodCol = 5;
 
    //设置当前高程
    floodAnalyzeInfo.Height = 330.43;
    //设置最大高程
    floodAnalyzeInfo.MaxHgh = 488.9689;
    //设置最小高程
    floodAnalyzeInfo.MinHgh = 330.43;
    //淹没区域扩大倍数
    floodAnalyzeInfo.RegZoom = 1;
    //数据模式控制，交互拾取和用户输入两种。 0：交互拾取模式； 1: 用户输入模式。
    floodAnalyzeInfo.DataType = 1;
 
    //设置观察方式、观察点、分析区域 
    Dot3D ObserveDot = new Dot3D(570424.75, 4749015.5, 330.43);
    Dot3D startdot = new Dot3D(570651.44, 4749119.50, 261.61);
    Dot3D enddot = new Dot3D(575740.31, 4752836, 330.57);              
 
    //设置观测点
    floodAnalyzeInfo.ObserveDot = ObserveDot;
    //设置分析区的起始点
    floodAnalyzeInfo.StartDot = startdot;
    //设置分析区的结束点
    floodAnalyzeInfo.EndDot = enddot;
 
    //创建地形分析工具
    TerrainAnalyzeTool floodAnalyzeTool = new TerrainAnalyzeTool(this.sceneCtrl);
    //设置洪水淹没信息
    floodAnalyzeTool.SetAnalyzeInfo(floodAnalyzeInfo);
 
    //开始分析
    floodAnalyzeTool.Start();
}
rasSet.Close();
svr.DisConnect();

 

SetMaxHeightOfColor 方法示例

本示例为在三维场景中根据色表生成图例；成功运行本示例，需先在Form中添加scenetrol控件。

C#

 

using MapGIS.Scene3D;

 

//定义地形分析工具

TerrainAnalyzeTool demAnalyzeTool = new TerrainAnalyzeTool(this.sceneCtrl);

//设置分层文件路径（该文件可通过色表编辑功能导出得到）

demAnalyzeTool.SetPathOfColorFile(@"D:\分层文件.txt");

//设置分层的最大值

       

 

demAnalyzeTool.SetMaxHeightOfColor(556);

//设置分层的最小值

demAnalyzeTool.SetMinHeightOfColor(78);

//设置图例显示位置

demAnalyzeTool.SetOverlayPosition(100, 200);    

  

TerrainAnalyzeInfo info = new TerrainAnalyzeInfo();

//设置分析类型为从文件读取色表信息显示图例

info.AnalyzeType = TerrainAnalyzeType.CutLineFormFile_Show;

demAnalyzeTool.SetAnalyzeInfo(info);

demAnalyzeTool.Start();

 

UpdateData 方法示例

此处代码主要介绍了洪水淹没演示的实现.

C#

 

 
using MapGIS.ScenetControll;
 //选区按钮点击事件
        private void button2_Click(object sender, EventArgs e)
        {
            viewShedTool.UpdateData();
            viewShedInfo.StartPos = 0;  //开始选择起点  
            viewShedInfo.StartReg = 1;  //开始选择淹没区  
            
            //默认可视域颜色
            if (this.comboBox1.Text == "")
            {
                viewShedInfo.ViewClr = 0;     //可视域颜色
            }
 
            else
                viewShedInfo.ViewClr = int.Parse(this.comboBox1.Text);
 
            //默认非可视域颜色
            if (this.comboBox2.Text == "") 
            {
                viewShedInfo.ShedClr = 256; //非可视域颜色
            }
            else
                viewShedInfo.ShedClr = int.Parse(this.comboBox2.Text); ; //非可视域颜色
 
            //默认观察高程
            if (this.textBox1.Text == "") 
            {
                viewShedInfo.Height = 20000; //观察点高程
            }
 
            else
                viewShedInfo.Height = int.Parse(this.textBox1.Text); //观察点高程
 
          //默认透明度
            if (this.textBox2.Text == "") 
            {
                viewShedInfo.Alpha = 0;
            }
 
            else         
            viewShedInfo.Alpha =int.Parse(this.textBox1.Text);   //透明度（ 0-1.0之间有效）
 
            viewShedTool.SetAnalyzeInfo(viewShedInfo);
            viewShedTool.Start();
        }

备注：此处代码主要是为了描述 viewShedTool.UpdateData()，完整实例参考2D3DManage.
