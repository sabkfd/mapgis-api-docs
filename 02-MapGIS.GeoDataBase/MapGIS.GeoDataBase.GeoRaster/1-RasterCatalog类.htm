示例

CreateAndOpenRasterCatalog方法示例

C#

 
using MapGIS.GeoDataBase.GeoRaster;
using MapGIS.GeoDataBase;
 
Server Svr = null;
DataBase gdb = null;
//连接数据源，打开数据库          
Svr = new Server();
bool rtn = Svr.Connect("MapGisLocal", "", "");
if (rtn)
{
    gdb = Svr.OpenGDB("示例数据");
    //初始化栅格目录对象
    RasterCatalog _RasterCatalog = new RasterCatalog(gdb);
    //创建并打开
    int flg = _RasterCatalog.CreateAndOpenRasterCatalog("ts1");
    if (flg > 0)
    {
        int id = _RasterCatalog.ClsID;
        MessageBox.Show("数据目录创建成功");
    }
    else
    {
        MessageBox.Show("数据目录创建失败");
    }
    //关闭数据
    _RasterCatalog.Close();
    gdb.Close();
    Svr.DisConnect();
}

备注

1.此示例代码需要在VS工程里的引用下面添加如下组件，分别为MapGIS.GeoDataBase.dll；

2.可以在平台安装目录下面的Program文件夹下面找到对应的组件。

//定义变量

{
    MessageBox.Show("数据源连接失败");
    return;
}

CRasCtg = new RasterCatalog(db);
 
//打开栅格目录
CRasCtg.Open("test");

{
    MessageBox.Show("InsertItem成功！");

else
{
