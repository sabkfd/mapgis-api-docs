Open(

   string filePath,

   RasAccessType accType

)

bool

成功/失败，true/false

打开栅格数据集

filePath: 文件路径

accType: 访问类型

注解:

filePath支持:

1.若采用无参构造函数构造,此时filePath可以为地理数据中栅格数据集URL(此时第二个参数accType无效)或者为本地影像数据绝对路径(可以不以"file:///"开头);

2.若采用带GDB的构造函数构造,此时filePath可以为地理数据库中栅格数据集类名或者为地理数据中栅格数据集URL.

     bool flg = _RasterDataSet.SetProj(_SrefData);

     {

         MessageBox.Show("地理坐标系设置成功");

     }

     else

     {

         MessageBox.Show("获取地理坐标系设置失败");

     }

     //关闭数据集

     _RasterDataSet.Close();

 }
