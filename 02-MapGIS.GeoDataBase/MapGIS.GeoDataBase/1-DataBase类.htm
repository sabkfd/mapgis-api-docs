Close方法示例

C#

using MapGIS.GeoDataBase;

 

//定义变量

 DataBase gdb = null;

 Server svr = new Server();

 //连接本地数据源

 bool rtn = svr.Connect("MapGisLocal", "", "");

 if (rtn)

 {

     //打开数据库

     gdb = svr.OpenGDB("sample");

     if (gdb != null)

     {

         //关闭数据库

         if (gdb.Close())

         {

             MessageBox.Show("关闭数据库成功！");

         }

         else

         {

             MessageBox.Show("关闭数据库失败！");

         }

     }

     else

     {

         MessageBox.Show("打开地理数据库失败！");

     }

     //断开数据源连接

     svr.DisConnect();

 }

 else

 {

     MessageBox.Show("连接数据源不成功！");

 }

备注

1.此示例代码需要在VS工程里的引用下面添加如下组件，分别为MapGIS.GeoDataBase.dll；

2.可以在平台安装目录下面的Program文件夹下面找到对应的组件。

GetXClass 方法示例

获取简单要素类对象，只有首先获取了该类对象，才可以根据名称或ID来打开该类

C#

using MapGIS.GeoDataBase;

DataBase gdb = null;

//通过URL打开数据库

gdb = DataBase.OpenByURL("GDBP://MapGISLocal/world");

if (gdb != null)

{

    //获取数据：设置要获取的数据类型

    SFeatureCls sfcls= gdb.GetXClass(XClsType.SFCls) as SFeatureCls;

    if (sfcls != null)

    {

        MessageBox.Show("获取类对象成功!");

    }

    else

    {

        MessageBox.Show("获取类对象失败！");

    }

    gdb.Close();

    gdb.Dispose();

}

备注

1.此示例代码需要在VS工程里的引用下面添加如下组件，分别为MapGIS.GeoDataBase.dll；

2.可以在平台安装目录下面的Program文件夹下面找到对应的组件。

 

GetXclses(XClsType,Int32) 方法示例

C#

using MapGIS.GeoDataBase;

 

//第一种：不指定数据集ID，即为0时，获取数据库中指定类型数据的ID列表。

Server svr = new Server();

bool r = svr.Connect("MapGISLocal", "", "");

DataBase gdb = svr.OpenGDB("world");

if (gdb != null)

{

    List<int> idlist = null;

    idlist = gdb.GetXclses(XClsType.SFCls, 0);

    if (idlist != null)

    {

        MessageBox.Show("该数据库下共有简单要素类" + idlist.Count + "个！");

    }

    else

    {

        MessageBox.Show("该数据库下没有简单要素类");

    }

    gdb.Close();

    gdb.Dispose();

}

 

 //第二种：获取指定数据集ID中的指定数据类型ID列表

 Server svr = new Server();

 bool rtn = svr.Connect("MapGISLocal", "", "");

 DataBase gdb = svr.OpenGDB("world");

 if (gdb != null)

 {

    //获取要素数据集ID为34中的简单要素类ID列表

    List<int>   idlist= gdb.GetXclses(XClsType.SFCls,34) ;

    if (idlist.Count > 0)

    {

       MessageBox.Show("指定数据集中存在简单要素类");

    }

    else

    {

     MessageBox.Show("指定数据集中不存在简单要素类");

     }           

   gdb.Close();

   gdb.Dispose();

}
