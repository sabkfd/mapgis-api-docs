示例

Append 方法示例

C#

using MapGIS.GeoDataBase;

 

//文件增长信息

FileExtendInfo msg = new FileExtendInfo();

//存储对象

DataStore dataStore = new DataStore();

bool rtn = false;

 

// 指定添加hdf的相关参数

msg.MaxFileSize = 0;

msg.IsExtendable = true;

msg.ExtendMode = 1;

msg.ExtendSize = 10; // 增长10%

string hdfName = "datastore" + ".hdf";

string hdfPath = @"D:\" + hdfName;

//添加文件

rtn = dataStore.Append(hdfPath, 100, msg);

备注

1.此示例代码需要在VS工程里的引用下面添加如下组件，分别为MapGIS.GeoDataBase.dll；

2.可以在平台安装目录下面的Program文件夹下面找到对应的组件。

Create 方法示例

创建瓦片文件，保存在磁盘中

C#

using MapGIS.GeoDataBase;

 

FileExtendInfo msg = new FileExtendInfo();

bool openFlag = true;

bool rtn = false;

DataStore dataStore = new DataStore();

// 指定创建hdf的相关参数

msg.MaxFileSize = 0;

msg.IsExtendable = true;

msg.ExtendMode = FileExtendMode.Percentage;

msg.ExtendSize = 10; // 增长10%

string hdfName = "datastore" + ".hdf";

string hdfPath = @"D:\" + hdfName;

rtn = dataStore.Create(hdfPath, 100, msg, openFlag);

 

if (rtn)

{

    MessageBox.Show("创建成功");

}

else

{

    MessageBox.Show("创建失败");

}

备注

1.此示例代码需要在VS工程里的引用下面添加如下组件，分别为MapGIS.GeoDataBase.dll；

2.可以在平台安装目录下面的Program文件夹下面找到对应的组件。
