AddSet 方法示例

通过本方法，可以将查询记录集中的内容添加到同类型的类中。要求记录集和类的属性结构相同。

C#

using MapGIS.GeoDataBase;

using MapGIS.GeoObjects.Att;

 

 

ObjectCls obcls = new ObjectCls();

if (obcls.Open("gdbp://MapGisLocal/测试数据库/ocls/省级行政区对象类"))

{

   //设置查询条件

   QueryDef def = new QueryDef();

   def.Filter = "面积>=1147320000000";

   RecordSet rcdset= obcls.Select(def);

 

   //创建新的对象类

   ObjectCls destObcls = new ObjectCls();

   if (destObcls.Create("gdbp://MapGisLocal/测试数据库/ocls/对象类结果", obcls.Fields) <= 0) return;                   

 

   if (destObcls.AddSet(rcdset))

       MessageBox.Show("成功！");

   else

       MessageBox.Show("失败！");

 

   obcls.Close();

}

备注

1.此示例代码需要在VS工程里的引用下面添加如下组件，分别为MapGIS.GeoDataBase.dll,MapGIS.GeoObjects.dll；

2.可以在平台安装目录下面的Program文件夹下面找到对应的组件。

 

Append(IGeometry,Record,GeomInfo) 方法示例

C#

using MapGIS.GeoDataBase;

using MapGIS.GeoObjects.Att;

 

//打开对象类

ObjectCls obcls = new ObjectCls();

if (obcls.Open("gdbp://MapGisLocal/测试数据库/ocls/对象类结果"))

{

    //获取对象类属性结构

    Fields fields = obcls.Fields;

    Field field = new Field();

 

    field.FieldName = "编号";

    field.FieldLength = 15;

    field.MskLength = 15;

    field.FieldType = FieldType.FldInt64;

    if (fields.AppendField(field) <= 0) return;

 

    Record rcd = new Record();

    rcd.Fields = fields;

    obcls.Fields = fields;

    if (!rcd.SetValue("编号", 1)) return;

 

    if (obcls.Append(null, rcd, null) > 0)

        MessageBox.Show("添加成功！");

    else

        MessageBox.Show("添加失败！");

 

    obcls.Close();

}

备注

1.对象类添加时只需要设置属性信息即可；不必设置几何信息与图形信息。

2.此示例代码需要在VS工程里的引用下面添加如下组件，为MapGIS.GeoDataBase.dll。

3.可以在平台安装目录下面的Program文件夹下面找到对应的组件。

GetAtt 方法示例

C#

using MapGIS.GeoDataBase;

using MapGIS.GeoObjects.Att;

//定义变量

ObjectCls objectCls = null;//对象类

Record record = null;//属性记录对象

Fields fields = null;

Field field = null;

//初始化源对象类

objectCls = new ObjectCls();

//打开源对象类图层

bool rtn = objectCls.Open("gdbp://MapGisLocal/DevelopDB/ocls/马尔可夫预测数据1");

if (rtn)

{

    //给ID赋值

    long ID = 4;

    //获取ID为4的对象的信息，对象类没有几何信息以及图形信息，故此方法的出口参数里只有第二个有效

    record = objectCls.GetAtt(ID);

    if (record != null)

    {

        //取得属性结构对象中的字段数目

        fields = record.Fields;

        int num = fields.Count; //取每个字段的值          

        for (int i = 0; i < num; i++)

        {

            field = fields.GetItem(i);

            string name = field.FieldName;

            FieldType type = field.FieldType;

            switch (type)

            {

                case FieldType.FldBinary:

                    object val = null;

                    val = record.get_FldVal(name);

                    break;

                case FieldType.FldBlob:

                    break;

                case FieldType.FldByte:

                    break;

                case FieldType.FldString:

                    break;

                    //.......

                    //.......

                case FieldType.FldMap:

                    object val1 = null;

                    val1 = record.get_FldVal(name);

                    break;

            }

        }

        MessageBox.Show("取属性成功");

    }

    else

    {
