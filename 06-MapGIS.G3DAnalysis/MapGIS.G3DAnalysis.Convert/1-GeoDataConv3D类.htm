示例

Convert 方法示例

C#

using MapGIS.Scene3D;

using MapGIS.GeoMap;

using MapGIS.GeoDataBase;

using MapGIS.GeoObjects.Geometry;

using MapGIS.G3DAnalysis.Convert;

 

 

//关联数据源

 Server svr = new Server();

 svr.Connect("mapgislocal", "", "");

 //设置导入模型的目的数据库

 DataBase db = svr.OpenGDB("TestDB");

 if (db != null)

 {

     //创建简单要素类对象，为模型导入到数据库的简单要素类

     SFeatureCls desSfcls = new SFeatureCls(db);

     //创建打开文件对话框

     OpenFileDialog openFilelDialog = new OpenFileDialog();

     //设置打开文件对话框的文件格式

     openFilelDialog.Filter = "*.3ds;*.obj;*.dae;*.osgb|*.3ds;*.obj;*.dae;*.osgb";

     if (openFilelDialog.ShowDialog() != DialogResult.OK) return;

     ////获取选中文件的路径

     string path = openFilelDialog.FileName;

     int index = path.LastIndexOf('\\');

     //截取模型名称

     string name = path.Substring(index + 1);

     //设置类类型为面类型

     GeomType type = GeomType.Surface;

     ////创建简单要素类，名为"导入的模型"

     int clsID = desSfcls.Create(name, type, 0, 0, null);

 

     //创建数据转换对象

     GeoDataConv3D covert3D = new GeoDataConv3D();

     //数据导入

     int result = covert3D.SetImpDataSrc(path, desSfcls);

     //设置数据转换参数,第一个参数是否将文件作为一个要素导入,缺省值为false，第二个参数是否将系统库中已存在的纹理覆盖,缺省值为true

     covert3D.SetOption(false, true);

     //设置导入模型的位置

     Dot3D posDot = new Dot3D(0, 0, 0);

     Dot3D scale = new Dot3D(1, 1, 1);

     Dot3D angle = new Dot3D(0, 0, 0);

     //设置数据变换属性 

     covert3D.SetTransOption(false, posDot, scale, angle);

     //开始转换

     int i = covert3D.Convert();

     if (i > 0) MessageBox.Show("导入成功");

     //结束转换

     covert3D.Close();

     desSfcls.Close();

     db.Close();

 }

 else

 {

     MessageBox.Show("数据库打开失败！");

    

 }          

 svr.DisConnect();

备注

1.通过本方法，可以将外部模型导入到MapGIS数据库中。

2.此示例代码需要在VS工程里的引用下面添加如下组件，分别为MapGIS.GeoDataBase.dll,MapGIS.GeoObjects.dll，MapGIS.GeoMap.dll，MapGIS.G3DAnalysis。

3.可以在平台安装目录下面的Program文件夹下面找到对应的组件。

 

CreateMcxFile 方法示例

本示例为将本地的倾斜影像数据转换成MCX文件。

C#

using MapGIS.G3DAnalysis.Convert;

using MapGIS.GeoObjects.Geometry；

 

Dot3D post = new Dot3D();

post.X = 0;

post.Y = 0;

post.Z = 0;

 

//执行数据转换

int i = GeoDataConv3D.CreateMcxFile(@"F:\data\3Ddata\倾斜摄影数据\测试数据\testData", @"F:\data\3Ddata\倾斜摄影数据\测试数据\testGD.mcx", post);

if (i>0)

{

    MessageBox.Show("转换成功！");

}

else

{

    MessageBox.Show("转换失败！");

}
