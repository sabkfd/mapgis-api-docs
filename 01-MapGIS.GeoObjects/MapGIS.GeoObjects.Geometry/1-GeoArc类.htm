示例

Buffer(Double,Double) 方法示例

C#

 
using MapGIS.GeoMap;
using MapGIS.GeoObjects.Geometry;
 
//定义弧段对象
GeoArc arc = new GeoArc();
//定义缓冲结果
IGeometry geom = null;
//定义左右缓冲区半径
double lDis = 0.5f;
double rDis = 0.5f;
            
//初始化弧段对象
arc.Set(new Dot(100, 100), 130, 45, 90);
            
//离散化
arc.DisperseToDots(2);           
//计算左右缓冲区
geom = arc.Buffer(lDis, rDis);
 
if (geom != null)
  {
       MessageBox.Show("计算左右缓冲区成功");
  }
else
  {
       MessageBox.Show("缓冲失败");
  }
 return;

备注：对该几何弧段进行缓冲区分析，左/右半径值单位与当前几何坐标一致，若分析成功，得到GeoPolygon类型对象。

 

CalLength(SRefData) 方法示例

示例说明：构造一个几何弧段，并计算该弧段在“北京54_度”坐标系下的实地长度。

C#

using MapGIS.GeoObjects.Geometry;

using MapGIS.GeoObjects.SpatialRef;

using MapGIS.GeoDataBase;

 

/*

    * 构建几何弧段

*/

GeoArc arc = new GeoArc();

//初始化弧段对象

arc.Set(new Dot(100, 100), 100, 10, 85);

//离散化

arc.DisperseToDots(2);

/*

    * 通过空间参照系管理对象

    * 获取空间参照系

    * 此处是北京54坐标系_度

*/

Server srv = new Server();

bool rtn = srv.Connect("mapgislocal", "", "");

if(!rtn)

{

   MessageBox.Show("数据源连接失败");

   return;

}

DataBase db = srv.OpenGDB("sample");

SRefData srefData = new SRefData();

SpatialRefMng srefMng = new SpatialRefMng(db);

//北京54_度

srefData = srefMng.Get(2);

//实地长度(单位米)

double lth = arc.CalLength(srefData);

 

Contains(IGeometry,Double) 方法示例

示例说明：构造一个几何弧段，并取几何弧段的第一个点创建一个点对象，判断几何弧段与这个点对象的包含关系。 

C#

using MapGIS.GeoObjects;

using MapGIS.GeoObjects.Geometry;

using MapGIS.GeoObjects.Info;     

//构造A圆弧

GeoArc arcA = new GeoArc();

//初始化弧段对象

arcA.Set(new Dot(150, 150), 100, 0, 180);

//离散化

arcA.DisperseToDots(2);        

//获取弧段A的第一个三维点对象

Dot3D dt=arcA.Get3Dots().GetItem(0);

//创建几何点B对象

GeoPoint geopnt = new GeoPoint(dt);

//验证弧段A与几何点B否是包含关系

bool ctns = arcA.Contains(geopnt, 0.0001);

Crosses(IGeometry) 方法示例

示例说明：构造一个几何弧段，一个多边形，判断几何弧段是否穿过多边形。

C#

using MapGIS.GeoObjects.Geometry;

using MapGIS.GeoMap;

using MapGIS.GISControl;

 

//定义一个MapControl控件对象

MapControl mapControl1 = new MapControl();

mapControl1.Dock = DockStyle.Fill;

this.panel1.Controls.Add(mapControl1); //把MapControl控件承载在Panel控件上面

 

//自定义绘图对象

Display disp = mapControl1.Display;

 

//构建矩形

GeoPolygon polygon = new GeoPolygon();

GeoVarLine varLine = new GeoVarLine();

//构建折线对象（矩形）

varLine.Append(new Dot(100, 100));

varLine.Append(new Dot(200, 100));

varLine.Append(new Dot(200, 200));

varLine.Append(new Dot(100, 200));

varLine.Append(new Dot(100, 100));

GeoLines lines = new GeoLines();

lines.Append(varLine);

polygon.Append(lines);

//绘制该矩形

disp.Begin();

disp.SetBrush(6, 0, 0, 0, 0);

disp.PolyPolyLine(lines);

disp.PolyGon(polygon);//绘制多边形

disp.End();

//构建弧段

GeoArc arc = new GeoArc();

//初始化弧段对象

arc.Set(new Dot(100, 100), 130, 45, 90);

//离散化

arc.DisperseToDots(2);

//开始绘制

disp.Begin();

disp.SetPen(1, 6);

disp.MoveTo(new Dot(100, 100));

//绘制弧段

disp.Arc(arc);

disp.End();

//判断穿过关系

bool cross = arc.Crosses(polygon);

Offset 方法示例

示例说明：构建圆弧A，将该圆弧A按照一定的偏移量偏移后得到圆弧B。

C#

 
using MapGIS.GeoObjects.Geometry;
using MapGIS.GeoMap;
using MapGIS.GISControl;
 
//定义一个MapControl控件对象
MapControl mapControl1 = new MapControl();
mapControl1.Dock = DockStyle.Fill;
this.panel1.Controls.Add(mapControl1); //把MapControl控件承载在Panel控件上面
 
//自定义绘图对象
Display disp = mapControl1.Display;
 
//构造A圆
GeoArc arcA = new GeoArc();
//初始化弧段对象
arcA.Set(new Dot(150, 150), 50, 0, 360);
//离散化
arcA.DisperseToDots(2);
//绘制圆A
disp.Begin();
disp.SetPen(3, 5);
disp.MoveTo(arcA.GetCenter());
disp.Arc(arcA);
disp.End();
//构造圆B
GeoArc arcB = null;
arcB = (GeoArc)arcA.Offset(150,150);//由圆A偏移构造圆B
//绘制圆B
Dot center = arcB.GetCenter();//获取圆B中心点
disp.Begin();
disp.MoveTo(center);
disp.SetPen(1,6);
disp.Arc(arcB);
disp.End();

备注：该偏移的参考坐标系是设备坐标系，即客户区左下角为原点，X轴正方向朝右，Y轴正方向朝上。若偏移参数offsetx取正值5，则表示向X正向平移5个像素。
