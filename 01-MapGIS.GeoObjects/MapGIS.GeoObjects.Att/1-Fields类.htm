示例

AppendField 方法示例

向示例数据"Ku6_1.wl添加一个属性字段"中地数码"

C#

 
using MapGIS.GeoDataBase;
using MapGIS.GeoObjects.Att;
 
//定义变量
Server srv = null;
DataBase db = null;
IVectorCls VecCls = null;
Fields Filds = null;
Field Fild = null;
 
//变量初始化
srv = new Server();
Fild = new Field();
 
//连接数据源，打开数据库
bool rtn = srv.Connect("MapGislocal", "", "");
if(!rtn)
{
   MessageBox.Show("连接数据源失败");
   return;
}
db = srv.OpenGDB("sample");
 
//打开简单要素类
VecCls = db.GetXClass(XClsType.SFCls) as IVectorCls;
VecCls.Open("Ku6_1.wl", 0);
 
//注意：这句代码不能丢，否则原来的字段会丢失
Filds = VecCls.Fields;
 
//设置要添加的字段
Fild.FieldName = "中地数码";
//字段长度
Fild.FieldLength = 255;
//字段类型
Fild.FieldType = FieldType.FldString;
//可编辑
Fild.Editable = 1;
//字段字符长度，必须设置
Fild.MskLength = 15;
 
//添加一个属性字段，字段名为：中地数码 
Filds.AppendField(Fild);
 
//对简单要素类设置属性
VecCls.Fields = Filds;
 
//关闭类、数据库、断开数据源连接
VecCls.Close();
db.Close();
srv.DisConnect();

备注：向一个已有属性字段的空间数据添加属性字段时，应先获取该空间数据已有属性结构，如上例中Filds = VecCls.Fields;再在已有属性结构对象基础上添加新字段，以防止已有属性结构丢失。

SetItem(Int32,Field) 方法示例

C#

 
using MapGIS.GeoDataBase;
using MapGIS.GeoObjects.Att;
 
//定义变量
Server srv = null;
DataBase db = null;
IVectorCls VecCls = null;
Fields Filds = null;
Field NewFild = null;
 
//变量初始化
srv = new Server();
 
//连接数据源，打开数据库
srv.Connect("MapGislocal", "", "");
db = srv.OpenGDB("sample");
 
//打开简单要素类
VecCls = db.GetXClass(XClsType.SFCls) as IVectorCls;
VecCls.Open("Ku6_1.wl", 0);
//注意：这句代码不能丢，否则原来的字段会丢失
Filds = VecCls.Fields;
//注意：防止该字段原来信息丢失
NewFild = Filds.GetItem(1);
//仅修改字段名称，该字段其它属性不变
NewFild.FieldName = "武汉中地数码";
//修改字段
Filds.SetItem(1, NewFild);
VecCls.Fields = Filds;
/**
    * 不能此种方式，修改会不成功
    * VecCls.Fields.SetItem(1, NewFild);
    */
VecCls.Close();
db.Close();
srv.DisConnect();

备注：在对字段进行删除、插入、修改操作时，需要保证该属性结构所在的简单要素类、对象类等未被占用状态，否则无法操作成功。
